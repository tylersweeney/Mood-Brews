Beer Me! provides information in an entertaining way while promoting local breweries. Data on customers can be used to enhance advertiser targeting. We use Face++ API to recognized facial details and cues from a user uploaded photo. The facial recognition data is cross referenced with BrewDB API to recommend a local brew and a map locator of local breweries that serve the recommended beer.
