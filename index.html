A Fun solution for making group decisions

Beer Me! provides information in an entertaining way while promoting local breweries. Data on customers can be used to enhance advertiser targeting. We use Face++ API to recognized facial details and cues from a user uploaded photo. The facial recognition data is cross referenced with BrewDB API to recommend a local brew and a map locator of local breweries that serve the recommended beer.


Beer me! uses HTML, CSS, and Javascript to creat a sleek UI. It uses jQuery, AJAX, Firebase, local storage and responsive design to deliver a polished functional product the entertains and provides useful information. Connecting local breweries to new customers, and making the world a better place.


Ideas for future features inclue user profiles, proximity chat, and local brewery trivia nights!
